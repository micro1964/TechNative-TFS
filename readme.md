<h1> Mini-project; web calculator </h1>

<p>This is a simple web-aclculator constructed using #HTML, #CSS, and #JavaScript.
It allows users to enter basic math expressions, calculate results by clicking on "calculate" button on the screen  and reset the calculator easily by clicking on the reset button.</p>


<h2> FILES Used </h2>

<p>index.html     # HTML structure</p>
<p>style.css      # Styling</p>
<p>script.js      # Calculator logic</p>
<p>README.md      # project documentation</p>



<h2> HOW TO USE </h2>

<p>Open index.html in your browser</p>
<p>Type a math expression into the input field</p>
<p>Example: 10 + 5 * (2 + 3)</p>
<p>Click Calculate.</p>
<p>Click Reset to clear everything.</p>

Mini-project; web calculator

This is a simple web-aclculator constructed using #HTML, #CSS, and #JavaScript.
It allows users to enter basic math expressions, calculate results by clicking on "calculate" button on the screen  and reset the calculator easily by clicking on the reset button.


FILES Used

index.html     # HTML structure
style.css      # Styling
script.js      # Calculator logic
README.md      # project documentation



HOW TO USE

Open index.html in your browser
Type a math expression into the input field
Example: 10 + 5 * (2 + 3)
Click Calculate.
Click Reset to clear everything.
